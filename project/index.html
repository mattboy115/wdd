Dropdown Nav Bars

WHAT I HOPE TO PRACTICE/LEARN

I have always had a lot of trouble with nav bars. They would never format right and I'd have a lot of trouble with their margins and padding and end up with a really messed up piece of garpage. I'd have to start over. Then when I started trying to make them with dropdowns using CSS it only got worse because then the dropdowns wouldn't work and when I got them to work, it wouldn't be possitioned right. I had to use my own tricks to get those things to work right. Then I learned how to use jQuery to make even better dropdowns a lot easier. But now I am hoping to unlearn all the bad practices I was using before.

HOW THIS RELATES TO WHAT I HAVE/WILL LEARN IN WDD.

Adding dropdowns to a navigation menu has a great deal to do with WDD. A navigation bar is the user's main way to get around a website. Adding expanding menues also expands the user's options as far as places they can get to from where they are.

How I Would Build This

  1. 1Start with a <nav> tag.
  2. 2Add a <ul> inside the <nav> and remove the list-stlye-type.
  3. 3Inside the <ul> add a <li> tag for each menu item and use styling to make them inline-block.
  4. 4Nest another <ul> inside each menu item that should have a dropdown.
  5. 5Use pseudo classes in your stylesheet to make the sub <ul> appear when the <li> are hovered over.
